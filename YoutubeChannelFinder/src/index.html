The YouTube Channel Finder

What is this application?

This application allows you to search for YouTube channels that are of interest to you. You can type whatever you want to search for, for example "Search for cats" or "I want to find channels about RC Hobbies", or simply "Hobbies".

How does this work?

This application uses Wit.ai to gather variables and data from any searched phrase, this is then sent into the YouTube Data API to search for channels tagged with those key words. In order to search you must log in through Google. Google require permissions based on your Google account connected to YouTube, these include accessing the users YouTube account, however due to the features used in this application I have kept it so that it only reads the users YouTube account, and doesn't change any data as this application just needs the user to login to access YouTubes data.
